# Java Basic Projects

This repository contains simple beginner-level Java programs.

## Projects Included
- **Calculator** – Performs basic arithmetic operations.
- **OddEven** – Checks whether a number is odd or even.
- **Temperature Converter** – Converts Celsius and Fahrenheit.
- **Number Rotation** – Rotates a number left or right based on input.
- **Simple Interest Calculator** – Calculates simple interest.

Each project is kept in a separate folder with its own Java file.
